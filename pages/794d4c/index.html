# Gitalk评论功能不生效

报错：

gitalk未找到相关的 Issues 进行评论解决方案
1

解决：

  1. GitHub中 OAuth Apps配置正确

    Setting -> Developer settings -> OAuth Apps -> new OAuth App

    配置完成后，即生成Client ID/Client secrets

    重点提示：Homepage URL，必须是HTTPS。如果不是HTTPS则会导致报错 gitalk未找到相关的 Issues 进行评论

  2. 创建GitHub Issue Label
