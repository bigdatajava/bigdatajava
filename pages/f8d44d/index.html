# 报错：命令行输入命令，报错：Similar command is: ‘xx’

现象举例：

source ~/.bashrc 
bash: tty: command not found...
Similar command is: 'tty'
1
2
3

解决：

[root@localhost mongodb]# export PATH=/bin:/usr/bin:$PATH
