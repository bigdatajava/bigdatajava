# 业务脑图：盘中交易、盘后结算、风控相关知识点

债券质押式回购为“每百元资金到期年收益”；

债券买断式回购为“每百元面值债券的到期购回价格”；

除权除息：因应发放股票股利或现增而向下调整股价就是除权，因应发放现金股利而向下调整股价就是除息 (opens new window)；

仓位：指投资者实有资金和实际投资的比例。例如：你有10万用于投资，但现在用了4万买股票或基金等，则你的仓位是40%；

利多：是刺激股价上涨的信息，如上市公司业绩好转，银行利率降低等对股价上涨有利的信息。利空则是和利多相反。

# 市价、限价

  1. 限价委托(limit order)
  • 在委托时对买卖的价格作出限定
  • 在买入股票时，限定最高价
  • 在卖出股票时， 限定最低价
  • 限价委托的最大特点是，股票的买卖可按照投资人希望的价格或者更好的价格成交，有利于投资人实现预期投资计划
  1. 市价委托(market order)
  • 在香托时只指定亦易数量而不给出具休的交易价格
  • 按该委托进入交易撮合系统时以市场上最好的价格进行交易
  • 市价委托的好处在于它能保证即时成交
  1. 市价委托规则

上海交易所-市价委托规则

  1. 最优五档即时成交剩余撤销 无需指定委托价格，委托进入交易市机时能与“最优五档”范围内的对手方队列成交或部分成交即予以撮合，未成交部分立即自动撤销的委托方式，一笔委托可以与对手方数笔不同价格的委托撮合成交
  2. 最优五档即时成交剩余转限价 无需指定委托价格，委托进入交易主机时能与“最优五档”范围内的对手方队列成交或部分成交即予以撮合，未成交部分自动转为限价的委托方式，一笔委托可以与对手方数笔不同价格的委托撮合成交

深圳交易所-市价委托规则

  1. 对手方最优价格 投资者无须指定委托价格当委托申报指令进入交易主机时，交易主机以当时集中申报簿有效竞价范围内对手方队列的最优价格为其指定价格的委托类型
  2. 本方最优价格 投资者无须指定委托价格，当委托申报指令进入交易主机时，交易主机以当时集中申报簿有效竞价范围内本方队列最优价格为其指定价格的委托类型
  3. 即时成交剩余撤销 无需指定委托价格，委托进入交易主机时能立即成交部分即予以撮合，未成交部分立即自动撤销的委托方式，一笔委托可以与对手方数笔不同价格的委托撮合成交
  4. 最优五档即时成交剩余撤销 与“即时成交剩余撤销委托”方式的区别就在于“即时成交剩余撤销委托”可以与对手方数笔不同价格的委托撮合，直至无有效对手盘，而“最优五档即时成交剩余撤销委托”只能与一定范围内的对手方队列成交
  5. 全额成交或撤销委托 无需指定委托价格，委托进入交易主机时能立即全部成交即予以撮合，否则委托全部撤销，该委托可以与对手方数笔不同价格的委托撮合成交

# 不受涨跌幅限制的几种情况

1.在股票名称前冠以“ST”字样的股票。也就是我们口中常说的垃圾股了。股票前面加ST，就代表着这家上市公司已经连续2年亏损或有着重大违规行为，交易所对该公司股票交易进行特别处理。它的交易日涨跌幅限制在5%。
2.新股上市首日，我国规定，新股上市的首日，它的价格不得高于发行价格的144%并且不得低于发行价格的64%。
3.股改股票完成股改,复牌首日，不受限制。
4.股改后的股票，达不到预计指标，追送股票上市当天不受限制。
5.某些重大资产重组股票比如合并之类的复牌当天不受限制。
6.退市股票恢复上市日不受限制。

# 价格最小单位

  1. A股、上海债券和债券买断式回购申报价格最小变动单位为0.01元；
  2. 基金、权证、深圳债券和质押式回购申报价格最小变动单位为0.001元
  3. B股交易上海为0.001美元，深圳为0.01港元上海债券质押式回购交易为0.005元
  4. 股票，基金涨跌幅比例为10%；
  5. 股票首个交易日无价格涨跌幅限制，单申报价格有限制；
  6. 科创板，创业板上市5日内无限制，5日后20%限制；

# 成交规则

价格优先：价格较高的买进申报优先于较低的买进申报，价格较低的卖出申报优先于较高的卖出申报；

时间优先：买卖方向，价格相同的，先申报者优先于后申报者。先后顺序按交易主机接受申报的时间确定；

# QFII，RQFII

QFII是指外资机构募集美元获批后，再转换为人民币直接投资A股；
RQFII（小QFII）是指境外人民币通过在香港的中资证券公司及基金公司投资A股。
QFII 机构部客户为境外基金机构，客户需求主要是交易股票、ETF等品种。客户根据业务需求发出指令，国内交易员接收指令进行执行。

# ETF

交易所交易基金（Exchange Traded fund），也称交易型开放式指数基金。
截止2016年12月30号，在中国上海证券交易所和深圳证券交易所上市的ETF共有140个（含货币基金ETF）。其中在上海证券交易所上市的有92个，交易代码都以51开头。在深圳证券交易所上市的ETF共有48个，交易代码都以15开头。


特点：
- 在二级市场买卖的开放式基金
- 可自由申购赎回的开放式基金
- 管理的资产是一篮子股票组合

规则：
- 当日申购的基金份额，同日可以卖出，但不得赎回
- 当日买入的基金份额，同日可以赎回，但不得卖出
- 当日赎回的证券，同日可以卖出，但不得用于申购
- 当然买入的证券，同日可以用于申购，但不得卖出

相关名词：
- ETF清单
- NAV（基金净值）
- IOPV（基金份额参考净值）
- 最小申赎单位
- 现金替代
- 现金替代溢价比例
- 现金替代比例上限
- 预估现金金额和实际现金差额

套利机制举例
- 比如，当上证50ETF的市场交易价格高于基金份额净值时，投资者可以买入组合证券，用此组合证券申购ETF基金份额，再将基金份额在二级市场卖出，从而赚取扣除交易成本后的差额。相反，当ETF市场价格低于净值时，投资者可以买入ETF，然后通过一级市场赎回，换取一篮子股票，再在A股市场将股票抛掉，赚取其中的差价。

# LOF

LOF基金（Listed open-ended fund）上市型开放式基金，是指通过深交所交易系统发行并上市交易的开放式基金。

供套利机会：
  - 溢价：基金净值《交易价格
  - 折价：基金净值》交易价格

LOF基金场内和场外转换:
- 认购:上市开放式基金发行期间在指定网点或交易所进行基金认购。
- 买卖:上市开放式基金发行结束后可以在交易所买卖该基金
- 申购赎回:投资者既也可以在指定网点申购与赎回基金份额
- 净值:申赎的价格，上市开放式基金收盘后发布。
- 跨市场转换:投资者如果是在指定网点申购的基金份额想要在交易所上网抛出，须办理一定的转登记手续;同样如果是在交易所网上买进的基金份额想要在指定网点赎回，也要办理一定的转登记手续。
- 场内股东账户:投资者在交易所场内使用的账户，深A股东账户或深圳基金账户。
- 场外基金账户:开放式基金账户，基金公司提供的(资金)账户。
- 费率(认购费、申购费、赎回费、转托管费、券商手续费):交易时设计的费用。

# 零股

所谓“零股”，即不足100股（最小交易单位）的股票。不到一个成交单位(一手=100股)的股票，如1股、10股，称为零股。
交易规则规定：在每个交易日的正常交易时间内，投资者可以委托卖出零股，但不能委托买入零股；在卖出股票时，可以用零股进行委托；
买进股票时不能以零股进行委托，最小单位是1手，即100股。投资者可以委托卖出零股，但不能委托买入零股。
